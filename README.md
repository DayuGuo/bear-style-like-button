## 功能
在 WordPress 的文章类型页面下方添加两个按钮：「点赞」和「支持」。

- 「点赞」：参考 Bearblog 点赞风格，以简洁版向上箭头为图标，配上一个轻柔的跳动动画。设置界面支持导出点赞数据。
- 「支持」：可在 WordPress 插件设置中修改链接，跳转到特定页面。

效果见本文下方，大家可以试试。

## 安装
本插件可在 Github 中 下载，并在 WordPress 后台上传安装。
